# NebulaBrowse 🌌

NebulaBrowse is a lightweight desktop web browser built using **Python**, **PyQt5**, and **cefpython3**. It combines the flexibility of Python with the power of Chromium Embedded Framework (CEF) to deliver a fast and modern browsing experience.

---

## 🚀 Features

- 🌐 Chromium-based browsing (via cefpython3)
- 🖥️ Native desktop GUI using PyQt5
- 🔎 URL navigation bar
- 🔄 Back / Forward / Reload controls
- ⚡ Fast rendering and modern web compatibility
- 🧩 Minimal and extendable architecture

---

## 📦 Requirements

Make sure you have the following installed:

- Python 3.7+
- PyQt5
- cefpython3

Install dependencies with:

```bash
pip install PyQt5 cefpython3
